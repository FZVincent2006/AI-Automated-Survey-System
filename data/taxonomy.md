# 学术文献分类树：LLM Agent 与相关领域前沿研究

## 1. Agent 核心技术与架构
本类别涵盖智能体（Agent）的基础能力、内部运行机制、记忆推理范式及底层架构设计，是实现自主智能的关键组成部分。

### 1.1 运行时与基础架构
关注Agent的执行环境、工具调用机制、可组合性及长期自主运行能力。
*   **代表性研究：** 
    *   **可组合运行时**：`HarnessX` 提出将 Agent harness 视为可组合原语的组装体，并通过进化引擎实现自适应优化；`AgentSpec` 则通过标准化接口对具身智能体的策略组件进行类型化组合研究。
    *   **长期自主与范式**：`From Chatbot to Digital Colleague` 定义了从聊天机器人到具备持久工作、自主推理和行动能力的“数字同事”的范式转变。
    *   **工具调用训练**：`CacheRL` 通过缓存执行轨迹和混合奖励训练小型代理模型学会调用工具的原因；`Agent over-reliance on tools` 则揭示了 LLM 对 GNN 工具的盲目服从现象。
    *   **递归代码代理**：`Recursive Agent Harnesses` 将递归单元视为完整的代码优先代理 harness，支持通过脚本并行生成子代理。

### 1.2 记忆与推理系统
研究Agent如何构建、存储、演化及利用记忆与推理轨迹。
*   **代表性研究：**
    *   **版本化推理记忆**：`GitOfThoughts` 将智能体推理树构建为 Git 仓库，支持对思考历史的提交、比对与合并。
    *   **流式记忆评测**：`StreamMemBench` 构建连续两步任务，以流式方式检验记忆系统对证据锚点的复用与反馈能力。
    *   **动态环境记忆演化**：`EvoArena` 记录记忆演化历史，让代理从记忆变化中推断环境演变，提升动态适应性。

### 1.3 通信与协作协议
研究多Agent之间的通信方式、信息共享策略与协作机制。
*   **代表性研究：**
    *   **文件基通信协议**：`tap` 利用文件（Markdown）作为持久通信媒介，结合独立 git 工作树，实现异构代理无共享内存的协作。
    *   **通信策略优化**：`Agent Communication Policy Optimization` 将通信行为形式化为可优化策略，利用文本与UI双通道互补性实现提示词级自我进化。
    *   **多智能体工作流适应**：`Graph-based Target Back-Propagation` 将工作流建模为 DAG，通过反向传播目标输出驱动分阶段提示更新。

### 1.4 技能进化与自我完善
研究Agent在无需外部真值反馈的情况下实现自我诊断、反思和技能迭代的能力。
*   **代表性研究：**
    *   **反思校准**：`Closing the Reflection Gap` 利用自身反思与环境真值的对比构造免费校准信号；`Retrospective Progress-Aware Self-Refinement` 让代理“先执行后反思”，回顾性生成进展信号。
    *   **无真值技能审计**：`SkillAudit` 通过成对轨迹审计隔离技能对行为的影响，实现无真值标签的技能诊断与改进。
    *   **自进化提问者**：`Self-evolving Visual Questioner` 使 VLM 通过自我生成并筛选问题实现自主进化。

## 2. Agent 评估、安全与可靠性
本类别涵盖对智能体性能的诊断评测、鲁棒性分析、安全约束及故障形成机制的研究。

### 2.1 基准与诊断
构建专门的评测框架，以揭示Agent在特定维度的能力差距与残余误差。
*   **代表性研究：**
    *   **能力差距评测**：`Running the Gauntlet` 引入时间感知、图形理解等未被充分探索的维度构建高难度基准。
    *   **对话能力基准**：`Dialogue SWE-Bench` 将对话能力视为编码性能的独立维度，设计对话式软件工程基准。
    *   **策略残留诊断**：`SANA` 通过消融理想组件，诊断 QA 代理在大规模数据湖上的策略级残余差距。
    *   **社会规范推理**：`LoSoNA` 构建群聊基准，迫使模型从示例中推断隐藏规范并生成恰当回应。

### 2.2 安全评估与防御
研究Agent在实际部署中的安全性，应对攻击拆解、医疗幻觉等风险。
*   **代表性研究：**
    *   **分解攻击**：`DECOMPBENCH` 揭示智能体在分解后的良性子任务面前安全防线大幅削弱的现象。
    *   **因果风险门控**：`Causal Risk Gating` 将能力最小化作为安全基元，通过分离因果风险与预测不确定性来决定是否推迟或放弃模型预测。
    *   **对抗审计防范**：`Trust but Verify` 采用事后对抗审计与多智能体反馈循环，在医疗推荐生成后验证其合规性。
    *   **规划失败检测**：`SIMMER` 构建符号世界模型和状态机执行器，显式检测 LLM 规划中的潜在失败与不可逆失效。

### 2.3 可靠性分析与故障诊断
聚焦于生产环境中故障的形成机制、可观测性问题及跨系统的真实度验证。
*   **代表性研究：**
    *   **无声故障与叙事**：`When Errors Become Narratives` 揭露“可故障化”现象，即 LLM 不仅隐藏错误还将其转化为看似合理的叙事。
    *   **城市模拟验证**：`When Plausible Is Not Realistic` 从移动定律、时间节奏等角度系统对比 LLM 代理移动模式与真实数据的差距。

### 2.4 约束策略与信任机制
研究受安全、预算等约束的策略优化，以及多智能体间的条件信任与攻击利用。
*   **代表性研究：**
    *   **效用约束 MDP**：`Utility-Constrained Policy Optimization` 通过效用约束实现安全策略训练后灵活调整约束限制。
    *   **网络防御约束**：`Safety-Contract Graph MARL` 将可复用运维预算显式建模到多智能体决策中。
    *   **技能条件信任**：`Conditional Trust in Agent Swarms` 分析技能条件信任在提升数据效率与被攻击者利用下的双重性质。

## 3. Agent 治理与可信人机交互
本类别关注AI治理政策、专家信任构建机制以及人机交互中的设计理念。

### 3.1 治理框架与政策
*   **代表性研究：**
    *   **开源政策对齐**：`Regulating the Machine Contributor` 通过比较多个开源组织政策与主要 AI 治理框架的对齐程度，建立政策成熟度的多维分类体系。

### 3.2 可信交互与溯源
*   **代表性研究：**
    *   **溯源优先设计**：`SpheriCity` 采用“溯源优先”设计视角，以证据可追溯性、结构化综合与交互支架作为赢得专家信任的核心机制。

## 4. Agent 应用与领域系统
本类别展示了Agent技术在特定垂直领域（如科学发现、自动驾驶、软件工程、医疗、教育）的落地应用与系统设计。

### 4.1 科学发现与知识工程
*   **代表性研究：**
    *   **知识图谱构建**：`Agents-K1` 将原始文档转化为代理原生科学知识图谱，实现端到端知识协调。
    *   **环境工程设计**：`EurekAgent` 提倡通过环境工程替代工作流工程，激发有效探索与系统化制品管理。
    *   **自动化形式化审计**：`Formalizing Numerical Analysis` 引入超越编译通过的多维质量审计，揭示代码编译掩盖的不忠实形式化模式；`Sorries Are Not the Hard Part` 则强调像审查软件一样审查形式化结果的可维护性。

### 4.2 具身智能与自动驾驶
*   **代表性研究：**
    *   **统一视觉导航**：`FloVerse` 以平面图先验为核心，统一 PointNav、ObjectNav 等多模态目标推理。
    *   **多智能体协同驾驶**：`Multi-Agent Embodied Autonomous Driving` 以共享世界模型统一视角，实现 V2X 观测向对齐状态与协调规划的转化。
    *   **机器人操作引导**：`Improving Robotic Generalist Policies` 利用流匹配的反转特性将次优动作引导至高质量模式；`Mana` 将灵巧操作重构为动画问题，生成可执行操作轨迹。

### 4.3 软件工程与代码智能
*   **代表性研究：**
    *   **规范代码基**：`No Accidental Software` 提出“代理优先”规范代码基底，通过折叠代码消除差异以摊销验证成本。
    *   **可控几何生成**：`VeriGeo` 将多模态组件和证明步骤表示为可执行动作序列，实现验证引导的反思修复。

### 4.4 医疗、能源与其他交叉学科
*   **代表性研究：**
    *   **隐私保护诊断**：`MedLatentDx` 在潜在空间进行多智能体通信，仅传递紧凑 KV 块以保护隐私。
    *   **LLM 医学编码**：`Can Post-Training Turn LLMs into Good Medical Coders?` 对比不同适应范式，揭示后训练的关键作用。
    *   **教育与学生模拟**：`Simulating Students' Programming Errors` 将 LLM 作为学生模拟器生成不可区分的合成编程错误。
    *   **氢能系统控制**：`A Statistical and Machine Learning Framework` 融合统计推断与机器学习，分层揭示线性驱动与非线性动态以优化控制。
    *   **无人机自主恢复**：`Selective Agentic Recovery` 引入学习型认知调用价值门控，按需调度远程大模型推理进行自主恢复。
