# LLM Agent 研究周报：架构凝练、可靠性反思与分类边界拓展

## 1. 本周研究动态总览
本周新增 49 篇论文，集中在四个主题：(1) **可靠性深层诊断**——`When Errors Become Narratives` 揭示生产环境“可故障化”无声失效，`SIMMER` 用符号世界模型检测规划不可逆失败，`DeCompBench` 暴露分解攻击下安全防线的溃缩；(2) **技能进化与自改进**——`SkillAudit` 通过成对轨迹审计实现无标签技能诊断，`RePro` 与 `RefGRPO` 分别从反思校准与回顾性进展信号拓宽自改进路径；(3) **通信与运行时工程**——`tap` 协议以文件介质实现异构代理无共享记忆协作，`HarnessX` 将 harness 视为可组合原语并配合进化引擎，`Recursive Agent Harnesses` 则将递归视为全栈代码代理单元；(4) **垂直领域深化**——医疗隐私（`MedLatentDx`）、科学知识协调（`Agents-K1`）、自动驾驶协同（综述）等方向持续渗透。

## 2. 核心技术路线演进
**工具调用与运行时降本**：`CacheRL` 引入缓存的执行轨迹和混合奖励，使小型模型学会工具调用的“原因”，但同时受限于缓存保真度；`Parallel-Synthesis` 直接在潜在空间合成并行分支，首 token 加速达 11 倍，却需下游微调，数据驱动属性明显。两极化趋势显示：效率优化仍依赖任务特定数据，通用性受限。

**记忆与推理架构**：`GitOfThoughts` 用版本控制管理推理树，却仅在高相似度案例上奏效，暴露记忆迁移的根本瓶颈；`EvoArena` 采用补丁记忆追踪环境演变，但绝对提升仅 1.5%，表明演化感知的推理提升远未充分。

**多智能体协作**：`tap` 的文件协议提供了故障可恢复的松耦合方案，但实验仅限双模型自应用；`GTBP` 利用 DAG 反向传播实现提示级收敛，其理论保证颇具吸引力，却缺乏开放动态环境的验证。总体而言，数据驱动的细粒度适配成为主流，但泛化性与计算开销的权衡依然尖锐。

## 3. 分类体系冲击与补充
新论文对既有 taxonomy 产生如下深化或交融：
- **1.1 运行时与基础架构**：`HarnessX` 的“组合 harness”和 `Recursive Agent Harnesses` 的递归代码单元将运行时抽象推向程序合成级，建议自此类可扩展运行时原语进一步细化。
- **1.2 记忆系统**：`StreamMemBench` 的流式记忆评测与 `EvoArena` 的记忆演化补丁推动记忆评估从静态转向动态连续，可形成新子类“演化记忆验证”。
- **1.3 通信协议**：`tap` 与 `MedLatentDx` 的潜在通信形成两极——前者是文件存储的显式协议，后者是隐私保护下的向量交换，二者可共同拓展为“异构安全通信”子类。
- **2.2 与 2.3 安全与可靠性**：`When Errors Become Narratives` 的五类无声故障分类法精准填补了 2.3 的空白；`Causal Risk Gating` 与 `DeCompBench` 分别引入因果决策与分解攻击视角，推动 2.2 向更有理论深度的防御范式演化。
- **4.4 医疗交叉**：`MedLatentDx` 的跨院诊断与 `Can Post-Training Turn LLMs into Good Medical Coders?` 的后训练编码实验为该类提供了“隐私-性能权衡”和“领域后训练”两个细粒度维度。
- **跨类融合**：`EurekAgent` 将环境工程设计替代工作流工程，实质上把代理治理与科学发现杂糅，提示未来的分类需包含“代理-环境共设计”的元层级。

## 4. 研究空白与未来方向
**（一）记忆迁移与抽象鸿沟**  
`GitOfThoughts` 证实记忆仅在近乎重复的实例上产生增益，更大模型也未能实现方法迁移。这暴露了以示例为单位的记忆存储缺乏结构化抽象，导致知识重用脆弱。未来必须探索如何将推理切片编译为可组合、可索引的抽象技能基元，而非依赖向量相似性。批判性在于：如果记忆体系仅在训练分布内有效，那么“自主学习”的承诺便大打折扣。

**（二）开放式安全验证与因果依赖的局限性**  
尽管 `SIMMER` 和 `Causal Risk Gating` 提供了局部安全保障，但前者限于可枚举的世界模型，后者依赖因果图的准确性；`DeCompBench` 揭示表面无害子任务可轻松瓦解安全壁垒，而当前防御多集中于事后审计或硬约束规划。研究空白是：自适应的安全形式化验证框架，能够应对分解粒度未知的攻击，且不捆绑特定因果假设。批判性在于：现有安全方案多假设已知威胁模型，而真实开放世界中威胁的进化速度远超防御的设计周期。

## 结论
本周研究表现出显著的架构工程深化与可靠性自觉，但记忆可迁移性、安全验证的开放性等领域仍存有根本性鸿沟。分类体系正在被跨层级融合所拉伸，亟需更灵活的多维元分类以包容代理与环境、模型与执行器之间的共同演化。
